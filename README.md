# 🌐 AXON Sovereign Language (AXL)
**The first programming language built on the principle of Logical Equilibrium.**

## ⚖️ What is AXON?
AXON is a Sovereign Logic System. In this language, every **Source** must have a **Manifest**. No logic is left unbalanced.

## 🚀 The Core: The Balance Engine
The current implementation features the **AXON-1.1 Core** written in Rust.
- **Integrity Checker:** A unique engine that calculates the 'Logic Weight' of the code.

## 🛠️ Quick Start
1. **Clone the repository:**
   `git clone https://github.com/mzn18117-code/AXON-Language.git`
2. **Run the engine:**
   `cargo run`
